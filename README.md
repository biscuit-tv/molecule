# molecule

:atom_symbol: Modern starter kit for React + Electron projects.

## Features

- Builds targets for both web and native
- React
- Redux
- Redux Saga
- React Router
- Hot reloading
- Webpack dashboard
- Optimized webpack builds (DLL and CommonsChunks)
- Immutable.js
- ES6
- Yarn
- Styled Component
